# Pangeo Julia Examples

Examples of using Pangeo cloud-based data with Julia and [Zarr.jl](https://github.com/meggart/Zarr.jl/issues/19#issuecomment-564522679).

### Notebooks

- `CMIP6-GCS-Julia.ipynb` [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/pangeo-data/pangeo-julia-examples/master?filepath=CMIP6-GCS-Julia.ipynb) Search the CMIP6 climate model data catalog on Google Cloud Storage and load and plot data.
